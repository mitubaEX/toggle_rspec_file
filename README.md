# toggle_rspec_file

toggle rspec file
(e.g.: app/controllers/user_controller.rb -> spec/requests/user_controller_spec.rb)

## Requirements

- Neovim nightly

## Installation

```
Plug 'mitubaEX/toggle_rspec_file'
```

## Usage

In ruby files.

```
:ToggleRspecFile
```

## Development

Load current repository files by bellow command.

```
nvim --cmd "set rtp+=."
```
